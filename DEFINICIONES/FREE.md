# FREE.md — Lead Magnet "Reactia Mini" (Paso 1 Hormozi)

## Resumen de Contexto (para IA / equipo)

**Oferta base (Reactia):** Programa 90 días B2B (5M COP) que combina estrategia + ejecución para dueños de negocio >5M COP/mes, 1+ año, validados. Entregables: Avatar, Oferta, Pricing, Landing, Playbook, CRM, Dashboard KPIs, Procesos, Roadmap. Diferencial: estrategia y ejecución simultáneas.

**Objetivo del Free:** Producto 100% software/automatizado (cero tiempo humano) que demuestra expertise, resuelve un dolor intenso inmediato, deja antojados sin suplir la necesidad completa, y **solo captura leads calificados** (que pueden pagar y cumplen perfil) para vender Reactia.

**Decisiones clave del usuario:**
- Core = **Diagnóstico Express simplificado** (12 preguntas versión reducida de la sesión de diagnóstico de 60 min de Reactia)
- Scoring con **IA (Openrouter barato / gratis si posible)** que suplanta la profundidad del diagnóstico humano pero en versión corta y concisa → "Generando tu Reactia Mini..."
- **Sin PDF, sin calls**: resultado vive en plataforma web propia, acceso temporal (7 días) → bloqueo suave + CTA a agendar diagnóstico estratégico
- **Gatekeeping obligatorio y filtrante** al inicio: facturación mensual + años operación + rol (dueño/CEO) → **si no cumple, no accede** (no perdemos tiempo con leads no calificados)
- **No email marketing** por ahora; capturar data y calificar leads internamente
- **Stack**: plataforma web propia (opensource/gratis), IA en backend para scoring, templates gated en la plataforma
- **Auth**: Gmail (OAuth) + WebAuthn (passkeys)
- **Métrica actual**: "tener algo bueno para dar gratis" (fase 1); métricas de conversión vienen en fase 2
- **Nombre**: fórmula MAGIC Hormozi + coherencia con Reactia → "Reactia Mini: Escáner de Crecimiento"
- **Filosofía**: "Te damos esto gratis para demostrar lo buenos que somos" (estilo Hormozi: valor extremo gratis → confianza → venta natural)
- **No decir "mismo diagnóstico"** → "adaptación express del diagnóstico Reactia"
- **Dashboard KPIs**: por definir en detalle luego; versión starter limitada en el free
- **Alcance free**: empieza mínimo, se amplía con feedback real

## Definición Final del Producto Free: **Reactia Mini: Escáner de Crecimiento**

### Qué ES (Entregable)
Una **herramienta web interactiva** (plataforma Kreanding) donde el dueño de negocio **calificado**:
1. **Gatekeeping filtrante** (30 seg): facturación mensual + años operación + rol (dueño/CEO) → **si no cumple ≥5M COP/mes, 1+ año, dueño/CEO → pantalla de "no califica" con contenido educativo genérico, no acceso a la herramienta**
2. **Autenticación**: Gmail OAuth + WebAuthn (passkeys) → sesión persistente
3. **12 preguntas simplificadas** (adaptación express del diagnóstico Reactia de 60 min) → 8-10 min
4. Pantalla: **"Generando tu Reactia Mini..."** (IA procesando respuestas con criterios propietarios)
5. **Resultado inmediato en plataforma** (acceso 7 días):
   - **Score por área** (0-100) con algoritmo IA propietario (Openrouter gpt-4o-mini / modelo local)
   - **Cuello de botella #1** identificado
   - **Próximo paso concreto** mapeado a mes Reactia ("Tu oferta no diferencia → Mes 1: Definiciones y Oferta")
   - **Benchmark anonimizado** vs negocios similares ("El 68% en tu etapa atascado en Oferta")
   - **Dashboard KPIs Starter** (versión limitada, solo lectura, métricas clave visuales — detalle por definir)
6. **CTA único prominente**: "Agenda tu Diagnóstico Estratégico sin costo" → Calendly/Cal.com → reunión de venta Reactia
7. **Día 7**: bloqueo suave (resultado visible pero "congelado", CTA persistente) → lead queda en CRM para seguimiento futuro

### Stack Técnico Recomendado (Opensource/Gratis)

| Capa | Herramienta | Por qué |
|------|-------------|---------|
| Frontend | **Next.js + Tailwind** (Vercel gratis) | Stack compartido con Reactia; SSR/SEO; componentes reutilizables |
| Formulario interactivo | **React Hook Form + Zod** (propio) | Control total UX, lógica condicional, cero coste, data propia |
| IA Scoring | **Openrouter (gpt-4o-mini / deepseek) / Ollama local** | Prompt estructurado con criterios Reactia; coste ~$0.0005/diagnóstico; fallback gratis |
| Base de datos + Auth | **Supabase (Postgres gratis 500MB)** | Auth (Gmail OAuth + WebAuthn), DB, edge functions, realtime; todo en uno |
| Dashboard KPIs starter | **Recharts / Tremor** (React) | Solo lectura, visual, embed en página resultado |
| CRM / Leads | **Supabase + NocoDB (Airtable open source)** | Tabla leads con score, respuestas, tags, calificación; gratis self-hosted |
| Despliegue | **Vercel (frontend) + Railway/Render (backend si hace falta)** | Gratis tier generoso; CI/CD automático |
| Analytics | **Umami (self-hosted en Railway)** | Privado, simple, gratis, eventos custom (completó diagnóstico, CTA click, etc.) |

**Coste estimado mensual: $0-3** (Openrouter ~$0.5-2/1000 diagnósticos; resto gratis).

### 12 Preguntas del Diagnóstico (Borrador — Adaptación Express del Diagnóstico Reactia 60 min)

| Área | Pregunta (simplificada) | Tipo | Peso IA |
|------|------------------------|------|---------|
| **Modelo** | 1. ¿Tu negocio vende producto, servicio, suscripción o mix? | Select | 10% |
| | 2. ¿Cuál es tu ticket promedio por cliente? | Number (COP) | 10% |
| **Oferta** | 3. ¿Tienes una oferta escrita y diferenciada que uses en ventas? | Sí/No/Parcial | 20% |
| | 4. ¿Sabes exactamente qué dolor resuelves y para quién (avatar)? | Escala 1-5 | 20% |
| **Clientes** | 5. ¿De dónde vienen la mayoría de tus clientes hoy? | Select (referidos, orgánico, ads, outbound, mix) | 10% |
| | 6. ¿Tienes un proceso definido para convertir leads en clientes? | Sí/No/Parcial | 10% |
| **Operaciones** | 7. ¿El dueño opera en el día a día o hay equipo delegando? | Select (yo todo, yo+ayudante, equipo pequeño, equipo con líderes) | 5% |
| | 8. ¿Tienes procesos documentados para las áreas clave? | Escala 1-5 | 5% |
| **Procesos** | 9. ¿Hay rituales semanales de seguimiento de métricas? | Sí/No | 5% |
